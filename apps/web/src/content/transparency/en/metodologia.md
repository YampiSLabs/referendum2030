---
The **Referendum 2030** system is governed by three irrevocable methodological principles:

1. **Uniqueness of vote:** Each test token is only usable once. Any reuse attempt is blocked by the server in milliseconds.
2. **Absolute decoupling:** Votes are stored in an aggregated totals board. Cryptographically, it is impossible to trace which token has cast which voting option.
3. **Open audit:** All token and voting transactions generate verifiable anonymous records on the public ledger in real time.
